## GETTING STARTED

* Clone the repository

```bash
git clone https://github.com/huynhthikimtrung321/html-css-training.git
```

* Move to folder `practice-two` 

```bash
cd html-css-training/practice-two
```

* Install packages

```bash
npm install
```

* Run the server

```bash
npm start
```
